# 🎓 Acadoscope: Insight to Impact
**Research Project by Division of Research & Technology - PPKU IPB 62**

## 📌 Overview
Proyek ini bertujuan untuk menganalisis kehidupan akademik mahasiswa, khususnya mengenai **"Hubungan Durasi Penggunaan Media Sosial TikTok dengan Tingkat Prokrastinasi Akademik"**.

## 🕵️‍♂️ Research Focus
* **Data Source:** Mahasiswa PPKU 62 IPB University.
* **Primary Metric:** Durasi Screen Time vs Nilai Akademik.
* **Goal:** Menemukan pola perilaku prokrastinasi untuk memberikan dampak positif bagi kehidupan mahasiswa.

## 🛠️ Tech Stack
* **Analysis:** Python (Pandas, Seaborn).
* **Environment:** Jupyter Notebook.
